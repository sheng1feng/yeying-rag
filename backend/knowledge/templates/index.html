知识库功能总览

1. 资产接入
通过受限 WebDAV `ak/sk` 接入 `warehouse`，只访问当前 Knowledge App 目录下被显式授权的资产。
2. 索引建库
按文件类型切块，生成 embedding，写入向量库，并维护导入任务明细与重试能力。
3. 知识项治理
支持候选知识项、人工知识项、修订与证据关联，避免退回 chunk 主叙事。
4. 发布与授权
以发布快照和服务授权作为稳定服务读面，并提供检索调试与审计能力。

API 接入文档

Step 1钱包 challenge / verify 登录 `knowledge`，获取 `knowledge JWT`。
Step 2在 `warehouse` 自行创建受限 `ak/sk`，再导入到 `knowledge` 作为读凭证或写凭证。
Step 3服务侧通过 `POST /service/search*` 读取已发布知识视图。
Step 4管理侧通过 `search-lab`、`retrieval-logs`、`source-governance` 调试命中与健康度。
Step 5如需发布变更，先在工作区更新知识项，再通过 `releases` 与 `grants` 控制读面。
交互式 OpenAPI：`/docs`。仓库文档：`docs/api-integration.md`。

产品操作手册

1登录钱包后，先导入读凭证与写凭证。
2在“资产仓库”中选择浏览凭证；上传操作只走写凭证。
3创建绑定源时选择对应读凭证，后续导入/重建按绑定源使用该凭证读取。
4完成 source scan、evidence build、candidate 生成与人工知识项整理。
5通过 release 发布稳定视图，再为服务创建 grant，并用 search lab 验证结果。
完整手册见 `docs/console-operations.md`。

检索可审计
